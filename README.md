# popquiz
Simple DevTest
This little index.html file calls the accompanying php file (hiding the function on the server away from prying eyes), which pulls
in data from the JSON feed at s://devtest.equisolve-dev.com, sorts it by reverse date, and sends it to the index file to display it
in the browser.  You can see it in action at http://www.itsnewstoyou.com/test
